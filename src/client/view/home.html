Thank you for your interest in tellem. This is a demonstration of the core functionality of tellem - real-time, channel-based notifications.

Using the Demo

The demo is currently running on development infrastructure, so avoid sending any personal data via tellem for the moment. Besides that, feel free to play around. Please note that the channels you currently see are shared amongst all demo users, so don't be alarmed if things change.

For the moment, you can create channels, subscribe/unsubscribe to those channels, and send bulletins. Hopefully it shows how tellem would be used within an organization or team.

Getting Started

Click some of the channels on the left for information on them. You can also create your own channels with the plus button.
