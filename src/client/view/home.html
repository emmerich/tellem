Thank you for your interest in tellem. This is a demonstration of the core functionality of tellem - real-time, channel-based notifications.

Using the Demo

The demo is currently running on development infrastructure, so avoid sending any personal data via tellem for the moment. Besides that, feel free to play around.

Please note that the channels you currently see are shared amongst all demo users, so don't be alarmed if things change.

There's a small utility to help when using the demo. It's a bot that can send bulletins to certain channels, giving you the opportunity to use tellem both as a sender and a receiver. It's called Tellie and can be found at the bottom right of the application.
